Comando var

Comando d!var


Sintaxe: d!var {variável} {valor}

Exemplo: d!var description A descrição de um servidor mágico!



Variáveis


Geral

Prefix - Define o prefixo do bot. (d!var prefix d!)

antiInvite - Ativa/Desativa o sistema anti convite de outros servidores do bot (d!var prefix 0/false/1/true)

antiUrl - Ativa/Desativa o sistema anti links do bot (d!var antiUrl 0/false/1/true)

description - Define a descrição do seu servidor do discord que virá na mensagem de bem-vindo/info (d!var description Esta é a descrição do meu servidor do discord!)

Sistema antiRaid

min_acc_days - Define o mínimo de tempo no qual uma conta do discord deve estar cadastrada para entrar no seu servidor em dias (d!var min_acc_days 5)

flood_control - Ativa/Desativa a inteligência artificial anti-flood. (d!var flood_control 0/false/1/true)

join_control - Define o número máximo de pessoas que podem entrar em 5 segundos

Sistema de Entrada/Saída

MANUTENÇÃO

joinMessage - Define o canal onde as mensagens de bem-vindo deverão ser enviadas (d!var joinMessage 0 = desligado /#bem-vindo(canal) = ativo)

joinMessage_type - Define o tipo de mensagem de entrada que o bot enviará (exemplo) (d!var joinMessage_type 3)

leaveMessage - Define o canal onde as mensagens de saída deverão ser enviadas (d!var leaveMessage 0 = desligado /#saiu(canal) = ativo)

leaveMessage_type - Define o tipo de mensagem de saída que o bot enviará (exemplo) (d!var leaveMessage_type 3)
